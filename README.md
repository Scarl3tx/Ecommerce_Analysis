# E-commerce Analysis Pipeline

## Overview
This project is an end-to-end e-commerce analytics pipeline using SQL and Python. The goal is to extract, transform, and analyze orders, payments, and customer behavior to produce actionable business insights.

## Dataset
- Publicly available **Olist Brazilian E-commerce dataset** (~100k orders)  
- Multi-table CSVs: `orders`, `order_items`, `order_payments`, `customers`, `products`, `sellers`, `order_reviews`, `geolocation`  
- Source: [Kaggle](https://www.kaggle.com/olistbr/brazilian-ecommerce)
